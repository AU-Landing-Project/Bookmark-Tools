== 1. Features ==
- Folders for users and for groups
- Group admin can control who can manage folders in a group
- Order bookmarks into folders
- Drag bookmarks onto folders
- Drag and drop folders to rearrange folder structure
- Delete a folder including subfolders (and optional all bookmarks in it)
- Update folder access on all subfolders (optional including all bookmarks) 
- Infinite folder structure (no limitations in length and depth)
- Breadcrumb on bookmark full view
- Take over of default bookmark menu options
